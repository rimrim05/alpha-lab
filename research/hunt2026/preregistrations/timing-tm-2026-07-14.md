# Preregistration — TM timing gate on the frozen attribution (EXP-2026-07-14-timing-tm)

Written and FROZEN 2026-07-14 before any timing regression is run. First mandatory

## Question
Beyond M2's factors (FF5 + MOM + TSMOM proxy + QQQ-residual), does any live book show
market-timing convexity — Treynor–Mazuy γ > 0 — in its claim-bearing blind window?

## Model (the one permitted extension; no other terms, lags, factors, or windows)
M2T: r_book − RF = α + Σβ_i·f_i(M2) + γ·(Mkt−RF)² + ε, daily, OLS, Newey–West lag 5
(the validated nw_ols with its self-check). Timing value TV = γ·V̄ann where V̄ann =
252·mean((Mkt−RF)²) over the same window — the standard TM contribution, annualized.
Estimation windows: each book's claim-bearing blind window exactly as frozen (hunt-1
books 2025-07-11→2026-05-29; hunt-2 books 2021-07-12→2026-05-29; FF truncation
2026-05-29). Full-history M2T reported as IN-SAMPLE context only.

## Known confound, handled by construction (pre-stated)
A naive trend rule has MECHANICAL TM convexity. M2 already includes the daily TSMOM
proxy (absorbs generic trend convexity); residual γ measures convexity beyond it. The
naive-parent comparison is still required: bench_qqq_sma200_2x runs through the same
M2T; a book only gets timing credit for γ EXCEEDING its naive parent's γ (vol/trend/
momentum books all compare against this one frozen parent — the only trend parent in
the spec registry). Volatility-timing convexity in vol space may not appear as
market-return convexity — a NOT-SUPPORTED verdict is therefore evidence against
TM-visible timing only; that limitation is pre-stated, not a rescue path.

## Controls (plumbing; hard gates)
- SPY and QQQ buy-and-hold: |t_γ| < 2 (a static book has no convexity).
- 1.5x static daily-rebalanced QQQ: |t_γ| < 2 (leverage alone adds none).
Control failure → fix plumbing, never thresholds.

## Gate (pre-committed, exhaustive; one verdict)
Per book, in its blind window: TIMING-POSITIVE iff (i) γ NW-t ≥ 2, (ii) TV > 0,
(iii) γ > γ_parent (bench_qqq_sma200_2x, same window), (iv) TV survives the financing
stress line (TV_stress = TV − max(avg_gross−1,0)·0 — financing hits levels not
convexity; instead require the book's M2T α + TV, stressed by the standard
alpha_stress haircut, > 0).
- SUPPORTED (strong): ≥1 book TIMING-POSITIVE at t ≥ 2.4 (Bonferroni m=3 clusters, as
  the frozen spec). → proceed to Phase 1/2.
- SUPPORTED (weak): ≥1 book TIMING-POSITIVE at 2 ≤ t < 2.4. → proceed, flagged.
- NOT SUPPORTED: no book TIMING-POSITIVE. → stop broad timing optimization; report
  "no evidence of TM-visible timing alpha for the existing signals"; the program's
  remaining permitted step is the negative-result memo.
Blind windows are claim-bearing; hunt-1 books' 219-obs window is LOW POWER (disclosed;
an underpowered miss is still NOT SUPPORTED under this gate — no post-hoc power
rescue).
